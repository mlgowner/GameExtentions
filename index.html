Гайд по установке

  1. Скачайте файл с сайта (Game Extensions.zip).
  2. Распакуйте архив в удобную папку.
  3. Откройте Roblox и используйте executor (например, Synapse X для скриптов).
  4. Вставьте скрипт или примените аватар в игре.
  5. Для мобильных: Установите через эмулятор, если требуется PC.

Системные требования: Windows 10+, Android 8+, минимум 4GB RAM.
